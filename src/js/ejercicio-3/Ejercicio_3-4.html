Torres de Hanoi

Esta aplicacion ayuda a resolver el juego de las Torres de Hanoi

Los tres palos se llaman A, B y C

El objetivo es pasar todos los discos al palo C. Las normas:
Se mueve de uno en uno y no se puede poner un disco mas grande encima de otro mas pequeno

El algoritmo esta disponible en Wikipedia

Para acabar el juego hay jugar en tres fases:

  • Mover n-1 discos de "A" a "B" usando "C" si se necesita
  • Mover el ultimo disco n de "A" a "C"
  • Mover n-1 discos de "B" a "C", usando "A" si se necesita.
